# 📂 File Organizer with Python

This script organizes files automatically into folders based on their type.

## ✨ Features
* **Wide Compatibility:** Supports images, videos, documents, and audio.
* **Smart Automation:** Automatically creates folders if they don't exist.
* **Flexible:** Easy to customize to your needs.

## 🚀 How to Use
1. **Change the path:** Open `main.py` and set your target directory route.
2. **Run the script:** Execute the file using Python in your terminal.
